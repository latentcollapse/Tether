# Tether v1.0 (Feb 2026)

Initial release.

- Content-addressed collapse/resolve with BLAKE3 + LC-B binary encoding
- SQLite persistence, in-memory transport, clipboard transport
- MCP server with core primitives: `tether_collapse`, `tether_resolve`, `tether_snapshot`, `tether_tables`, `tether_export`, `tether_import`
- CLI: `tether collapse`, `tether resolve`, `tether snapshot`, `tether tables`, `tether export`, `tether import`
- Python API: `SQLiteRuntime`
- Notifications convention with read receipts
